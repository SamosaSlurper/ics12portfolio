Array List

Assignment Intro

This unit on array lists was by far probably the hardest concept we learned this year. I say this as it was a new topic that required a lot of focus to understand initially. The assignment that I'll use to demonstrate my understanding is the Crazy Sentences and Poem assignment. To do this we utilized different array list methods, learned how to initialize array lists and came up with a good number of different parts of speech before making a randomly generated haiku poem out of these words.

Programming Concept

The concept behind array lists is quite simple. After you initialize by saying: 

ArrayList<variableType> variableName = new ArrayList<>(variableLength);

You can use array list methods such as .add, .remove, .set etc. to add new elements to the array list. Array lists are very convenient as they can store information in an organized manner while also allowing the user to easily access it.

Challenges

This was by far, one of the most annoying assignments all year. While creating the array lists and generating a randomized sentence was relatively easy, creating the haiku while making grammatical sense was difficult. Not only was I unsure of what to do for some parts, my program kept crashing for reasons that I didn't understand.

Solutions

To solve the problems that arose from this assignment, I utilized external help in geeks for geeks and oracle to help create the loop for each of the three lines of the haiku while also keep the syllable count in check. To do this, I assigned a syllable count to each word in the array lists and arranged them in such a way that it would always meet the requirements.

Reflection

In hindsight, the concept itself seems pretty easy but at the time, it was a little hard to understand and use. I am proud of how I managed to use external resources to help me better understand the concept and make the haiku which in the long run helped me study for my AP exam.
